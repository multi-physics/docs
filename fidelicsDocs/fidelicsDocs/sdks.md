## 灵活部署到研发环境
无论您是需要完全隔离的内网环境，还是希望快速验证的云端实例，我们都提供最自由的部署方式。

- **零依赖私有化部署**：单机/集群均可一键安装，无需任何第三方商业许可
- **全平台支持**：Windows / Linux / macOS 原生运行
- **Docker 官方镜像**：即将上线，支持 Docker Compose 与 Kubernetes 编排
- **SaaS 云托管版**（开发中）：按需付费，即开即用，适合快速原型验证
- **可扩展插件架构**：支持热插拔模块，业务代码与核心求解器完全解耦

> 您的代码、您的数据、您的集群 —— 完全掌握在自己手中

## 快速构建工业级多物理场方案
从单物理场到复杂的多物理场强耦合，数分钟即可完成方案搭建。

- **模块化拖拽式建模**：流体、固体、电磁、热、化学反应等 20+ 物理场模块
- **预置工业场景模板**：
  - 飞行器气动热力耦合
  - 下一代低噪音设计
  - 新能源电机电磁-热-流耦合
- **可视化耦合流程编排**：类似 Simulink 的图形化耦合接口，零代码完成复杂流程
- **分钟级从 0 到 1**：导入几何 → 选择模板 → 设置边界 → 开始计算

## 自由定义模型方程
真正的“开放”——您可以编写任何想实现的物理模型。

- **开放 PDE 接口**：支持标量、向量等丰富偏微分方程
- **Python / C++ 双语言扩展**：
  - Python：快速原型开发（推荐科研用户）
  - C++：极致性能（推荐工业用户）
- **自动有限元/有限体积离散**：只需写出弱形式或守恒形式，离散+组装全自动，顶级数值算法赋能，大幅提升仿真性能和体验
- **自定义源项、边界条件、材料模型**：所见即所得，即写即用
