# Fidelics —— 可扩展统一多物理场仿真平台

Fidelics 是一款可扩展的多物理场仿真平台，在统一的框架下提供下列一个或多个（支持耦合）物理场能力：

- 低马赫数不可压缩流动（low mach incompressible flow）
- 亚声速、跨声速、超声速可压缩流动（subsonic, transonic and supersonic compressible flow）
- 多相流（multiphase flow）
- 多组分流动（multispecies flow）
- 热动力学（thermal-dynamics）
- 气动声学（aerodynamic acoustics）
- 线性静力学与动力学结构力学（linear static and dynamic structural mechanics）
