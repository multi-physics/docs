Generalized CAE products

SIMX: MultiPhysics Simulation Software

SIMX has rich simulation and analysis capabilities related to fluids, solids, and heat transfer, and is one of the most advanced CAE software in China.

ModelX: Modeling Software for CAX

ModelX is a modeling environment specially developed for CAE. Its unique modeling capabilities can well unleash the application potential of CAE in modern R&D projects.

MeshX: Intelligent Meshing Software

MeshX is a modern meshing tool that, when combined with ModelX, can efficiently generate high-quality computational meshes. At the same time, the mesh scheme for special scenes can be implemented for geometry and mesh parameters, which can greatly improve the analysis efficiency of similar scenes.

DesignX: Modern 3D Design Software with CAE strongly coupled

DesignX is a 3D design software that uses CAE extensively. It takes advantage of the modeling function of Modelx, the multi-physics analysis function of SIMX and an efficient collaborative working mechanism. It exerts strong energy on product prototyping, optimization design and performance verification and evaluation.
